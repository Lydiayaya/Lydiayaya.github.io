hexo本地启动:hexo s

hexo清楚缓存：hexo clean

hexo 新建文件：hexo n “文件名”

hexo 生成网站静态文件到默认设置的public文件：hexo g（用于手动部署网站）

hexo部署本地代码到远端：hexo d

hexo 生成新页面：hexo n page name

hexo 草稿：hexo new draft “new draft”
